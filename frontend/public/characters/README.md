# Character Images

Add character profile photos here with the following names:

- `michael.jpg` - Michael Scott
- `jim.jpg` - Jim Halpert
- `angela.jpg` - Angela Martin
- `stanley.jpg` - Stanley Hudson
- `phyllis.jpg` - Phyllis Vance
- `toby.jpg` - Toby Flenderson

## Image Requirements
- Format: JPG, PNG, or WebP
- Recommended size: 128x128 pixels (or larger, will be scaled down)
- Square aspect ratio works best

## Where to Get Images
1. Google Images: Search for "[character name] the office profile"
2. The Office Wiki: https://theoffice.fandom.com/
3. NBC Official Site
4. Screenshot from the show

## Fallback
If an image is not found, the app will display initials with a colored background.
